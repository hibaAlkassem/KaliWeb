== Description ==

This is a Highly Customisable Cookie Banner Plugin with 18 customisable options enabling you to change the banner: 
<br />
<ul>
<li> - Position</li>
<li> - Size</li>
<li> - Banner Background colour</li>
<li> - Banner Text Colour</li>
<li> - Button Background Colour</li>
<li> - Button Text Colour</li>
<li> - Button Border Colour</li>
<li> - Background Hover Colour</li>
<li> - Background Hover Text Colour</li>
<li> - Button Background Hover Colour</li>
<li> - Button Text Hover Colour</li>
<li> - Button Border Hover Colour</li>
<li> - Responsive Width Break Size</li>
<li> - Responsive Width Break Size</li>
<li> - Delay time before fade out (if required)</li>
<li> - Fade out animation time</li>
<li> - Choice to use inline/external stylesheets and scripts</li>
<li> - Choice to include external script in header/footer</li>
<li> - Plus more...</li>
</ul>
The plugin is fully responsive if you have the viewport meta tag on your site, using media queries with the widths defined by you.
<br /><br />
In addition, output can be included in various way, either using external or internal scripts and styles with the option to include the JavaScript in the header or footer.
<br /><br />
All output is now minified by the plug-in meaning a 50% saving in file sizes. The included CSS and JS files are now less than 1.5kb in size combined.
<br /><br />
It also includes the following functionality:
<ul>
<li> - WPML/Polylang Support</li>
<li> - International translation ready (awaiting translations)</li>
</ul>

== Installation ==

1. Upload `cookie-banner` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Set your options from the Cookie Banner menu

Alternatively:

1. Login to your WordPress admin area
2. Search for `Cookie Banner`
3. Install `Cookie Banner`
4. Activate through the `Plugins` menu in WordPress or when asked during installation
5. Set your options from the Cookie Banner menu

== Frequently Asked Questions ==

= Why is the plugin stripping out my <a href=""></a> tags? =

- Only the &lt;a&gt; HTML tag is accepted in the Banner Text Plugin. To enable it to work properly, please remember to use single quotes (') rather than double quotes (") in your href tags.
